# kt-component-statemachine

## 作用
状态机组件，可动态编排业务逻辑，把"状态"转换交给状态机，开发人员只需要关注业务本身。应用场景如：订单、工单等有大量状态转移的场景。

## 模块介绍

| 模块                             | 作用       |     |
|--------------------------------|----------|-----|
| kt-component-statemachine-core | 状态机核心逻辑  |     |
| kt-component-statemachine-dao  | 状态机持久化操作 |     |
